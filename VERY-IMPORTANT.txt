- Max 500 tokens per response (cost-efficient)
- Rate limit: 20 AI messages per user per day
- System prompt includes platform knowledge + user context
- Falls back to pattern-based auto-replies if:
  - ANTHROPIC_API_KEY not set
  - API call fails (timeout, error)
  - Rate limit exceeded

Integration: src/app/api/chat/conversations/[id]/messages/route.ts
- When CLIPPER sends message:
  1. Checks needsHumanSupport flag (skips AI if true)
  2. Tries AI chatbot with last 10 messages as context
  3. Falls back to pattern-based auto-reply if AI fails
  4. Checks if AI suggests transfer (3 consecutive unable-to-help)
  5. If transfer needed: sets needsHumanSupport=true, notifies owners
- When OWNER/ADMIN sends message: clears needsHumanSupport flag

Schema change: added needsHumanSupport Boolean to Conversation model.

Estimated cost: ~$0.001 per AI response (500 tokens at Sonnet pricing).
