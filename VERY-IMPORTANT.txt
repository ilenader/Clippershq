4. MANUAL ACTIONS REQUIRED BY OWNER
=====================================

A. SET UP EMAIL (Resend):
   1. Go to https://resend.com and sign up
   2. Add your domain (clipershq.com) and verify DNS records
   3. Create an API key
   4. In Vercel → Settings → Environment Variables, add:
      - EMAIL_API_KEY = your Resend API key
      - EMAIL_FROM = "Clippers HQ <noreply@clipershq.com>"
   5. Redeploy

B. SET UP CRON BACKUP (cron-job.org):
   1. Go to https://cron-job.org and create account
   2. Create cron job:
      URL: https://clipershq.com/api/cron/tracking
      Schedule: Every 1 hour
      Method: GET
      Header: Authorization: Bearer [your CRON_SECRET value]
   3. Save and enable

C. SET UP YOUTUBE TRACKING (optional, requires API key):
   1. Go to https://console.developers.google.com
   2. Create project → Enable YouTube Data API v3
   3. Create API key
   4. Add YOUTUBE_API_KEY to Vercel env vars
   5. YouTube tracking is NOT yet implemented in code
      → Need to add YouTube actor to src/lib/apify.ts
      → Or build YouTube Data API v3 integration
   STATUS: YouTube clips can be submitted but views are NOT tracked

D. CREATE SUPABASE STORAGE BUCKET:
   1. Go to Supabase Dashboard → Storage
   2. Create bucket named "uploads"
   3. Set to PUBLIC
   4. This enables campaign image uploads from the admin panel

E. VERIFY DISCORD OAuth:
   1. Go to https://discord.com/developers/applications
   2. Verify redirect URL is: https://clipershq.com/api/auth/callback/discord
   3. If not, add it under OAuth2 → Redirects
