BUILD SCRIPT UPDATE — Summary
===============================

File: package.json
Changed: "build": "next build"
To:      "build": "prisma generate && next build"

Why: On Vercel, the src/generated/prisma/ folder is not committed to git
(it's in .gitignore). Without prisma generate in the build script, Vercel
would try to build without the generated Prisma client and fail.

Now the build pipeline is:
1. prisma generate → creates src/generated/prisma/ with typed client
2. next build → compiles the app (which imports from @/generated/prisma)

Verified: npm run build outputs:

  ✓ Compiled successfully
  ✓ Generating static pages (33/33)
